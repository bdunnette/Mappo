## Introduction

Mappo is a web application based on Meteor (http://meteor.com/) to edit maps in real time.

## Roadmap

1. Choice of background layers.
2. Integration with Open Street Map.
3. Searching.
4. Print and export maps.
5. Adding notes.
6. Online chat.
7. Authentication.
